# AI Prompt Evaluation Framework 🤖

A modular Python-based framework built to score AI-generated responses across 6 core alignment dimensions: factual accuracy, hallucination severity, reasoning depth, instruction compliance, coherence, and safety.

## 🔑 Key Features
- **Automated Comparative Logic:** Evaluates and ranks side-by-side responses from multiple models to mimic professional Anthropic and OpenAI workflows.
- **Structured Output Batching:** Processes batches of 50+ responses, outputting organized data into JSON and Markdown for direct ingestion into RLHF training pipelines.

## 🛠️ Tech Stack
- Python
- JSON Validation
- RLHF Evaluation Methodology
